# GenAI Trust Assistant

A decentralized AI verification assistant powered by GenLayer Intelligent Contracts.

## Overview

GenAI Trust Assistant is a prototype project that demonstrates how GenLayer can be used to create trusted AI workflows.

The system allows users to submit AI-generated information and receive a verification result through Intelligent Contract logic.

## Features

- AI content verification workflow
- Trust score calculation
- Verification history storage
- Intelligent Contract based decision process
- Transparent validation architecture

## How It Works

1. User submits AI-generated content.
2. Intelligent Contract receives the verification request.
3. Validators analyze the information.
4. The contract calculates a trust score.
5. The verification result is stored.

## Technology Stack

- GenLayer Intelligent Contracts
- Python
- AI Validation Workflow
- Web3 Architecture

## Project Goal

The goal of this project is to explore how decentralized AI verification can improve trust and transparency in AI-generated information.
